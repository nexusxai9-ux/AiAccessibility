# AI-Powered Assistive Computer Control System

## 🚀 Overview

This project is an AI-powered assistive system designed to enable people with upper-limb disabilities to use computers independently. By combining voice control, eye-tracking, and intelligent automation, the system removes the need for physical interaction with devices.

---

## ❗ Problem

People with upper-limb disabilities struggle to perform basic computer tasks like typing, clicking, and navigation. Existing solutions are often expensive, complex, or limited, leading to dependency on others.

---

## 💡 Solution

This system provides a unified, hands-free computer interaction experience using:

* 🎤 **Voice Control** – Execute commands and dictate text naturally
* 👁️ **Eye Tracking** – Move cursor and interact using gaze
* 🤖 **AI Assistance** – Automate tasks and handle multi-step actions
* 🤖 Autonomous AI System – Understands intent, processes context, and executes multi-step tasks independently

🧠 Neuroscience-Inspired Interaction – Designed around how humans naturally communicate (speech + vision), reducing cognitive load and learning curve

---

## ⚙️ Features

* Hands-free computer control
* Natural language command execution
* Cursor movement via eye tracking
* Task automation (emails, browsing, apps)
* Minimal and accessible user interface
* Scalable across devices and platforms

---

## 🧠 How It Works

1. User gives a voice command
2. AI processes and understands the request
3. Eye tracking controls cursor movement
4. System executes the task
5. Feedback is shown to the user

---

## ▶️ Running the MVP

To run the minimum working prototype (MVP), ensure all core modules are active:

* `ai.py` → Handles AI processing and task execution
* `ui.py` → Runs the user interface
* `dictation.py` → Manages voice input and speech-to-text
* `eye_cursor.py` → Controls cursor using eye tracking
* `open.py` → Handles app launching and system actions

### Steps:

1. Open your project directory
2. Run each module (preferably in separate terminals or integrated setup)
3. Ensure microphone and camera permissions are enabled
4. Start interacting using voice and eye movement
